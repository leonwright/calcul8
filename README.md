# Getting Started with Calcul8

Calul8 is a simple HTML based calculator. It is a simple calculator that can be used to perform simple calculations.

## How to use Calcul8

### Starting the application

```bash
$ cd /path/to/calcul8
$ npm start
```

### Running tests

```bash
$ npm test
```

### Running tests with coverage

```bash
$ npm test -- --coverage
```
